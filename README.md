# ENVS-193DS_homework-03
Homework 3 for Env 193DS at UCSB

## General information

This repository contains the Homework 3 assignment for the course ENVS 193DS at UCSB. The assignment includes an analyses of giant kelp frond elongation rate, updated personal data visualizations, an affective visualization draft, and a statistical critique of a table from the published paper, "Integrative ecological health assessments of an acid mine stream and in situ pilot tests for wastewater treatments" accesed from the ScienceDirect database.

## Data and file information

The `code` folder contains the Quarto file used to write and render the HW 3 assignment.

The `data` folder contains the data files used in the assignment, including that of ocean temperature and kelp elongation data, as well as my personal data.

The `images` folder contains images and PDF files used in the assignment, including my affective visualization sketch, draft, slides, and the table used for the statistical critique.

## Rendered output

The rendered homework output is included as a PDF generated from the Quarto document. The final PDF contains the written responses, annotated R code, code output, figures, captions, the affective visualization materials, and finally the statistical critique.
